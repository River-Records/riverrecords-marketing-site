# Stream × NNPEN — partnership one-pager

A clean, co-branded single page (US Letter, portrait) to share with NNPEN and
their members. Co-branded with the River Records mark and the NNPEN logo, both
pulled from `public/`. Use approved by NNPEN.

## Files
- **`stream-nnpen-onepager.pdf`** — shareable / print-ready.
- **`stream-nnpen-onepager.png`** — high-res render (200 dpi) for email, Slack, or slides.
- **`stream-nnpen-qr.png`** — standalone QR (intermediate, regenerated by the script).
- **`generate_onepager.py`** — regenerates everything.

## What the QR encodes
```
https://www.riverrecords.ai/np?utm_source=nnpen&utm_medium=onepager&utm_campaign=nnpen-partnership
```
Scans land on the `/np` page and are attributable in GTM/GA under
`utm_campaign=nnpen-partnership`, separate from AANP/QR-handout traffic.

## Regenerate / edit
```bash
pip install "qrcode[pil]" reportlab pymupdf
python generate_onepager.py
```
Copy/headline/offer text and layout live near the top of `generate_onepager.py`
(`URL`, `CAPS`, and the `build()` function). If the logos change, drop the new
files into `public/` and rerun.
